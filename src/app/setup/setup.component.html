Offline setup

1. Pull from online (this machine reaches both)

Your local API must set ONLINE_API_URL and the same ENROLLMENT_TOKEN as the online server (or trusted local-only: PROVISION_ALLOW_EMPTY_ENROLLMENT=true with no token). Uses POST /provision/pull-and-import.

2. Download bundle, then import locally

Use these on any PC that can reach the online API. Save the JSON file, copy it to the offline machine, then use section 3. The token is sent in the X-Enrollment-Token header (not in the URL).

Enter online base URL and owner email to enable copy buttons.
